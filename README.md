# image_processing

Description.
The package image_processing is used to: <br>
-	Processing: <br>
		- Histogram matching  <br>
		- Structural similarity <br>
		- Resize image
-	Utils: <br>
		- Read image <br>
		- Save image <br>
		- Plot image <br>
		- Plot result <br>
		- Plot histogram

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install image_processing

```bash
pip install image_processing
```
